# 🐾 Prototipação PetEncontra

**Documentação completa da prototipação de interface** para a plataforma de adoção de pets PetEncontra.

---

## 🎯 Visão Geral

Prototipação completa com **wireframes**, **protótipos** e **fluxos de usuário** para dois públicos:

- **Adotantes**: Buscar e filtrar pets disponíveis  
- **ONGs/Pessoas físicas**: Gerenciar pets para adoção

---

## 🖼️ Telas Protótipos

### **Tela 1 - Home (Página Principal)**

**Funcionalidades:**
- Header: Logo + menu (Início/Login/Cadastrar)
- Filtros: Espécie, Idade, Porte, CEP  
- Cards de pets com "Ver detalhes"
- "Encontrar pets perto de mim"

### **Tela 2 - Detalhe do Pet**

**Funcionalidades:**
- Foto ampliada + informações do pet
- Contato da ONG (telefone/email)
- Botões "Ligar" e "Enviar email"

### **Tela 3 - Login ONG**

**Funcionalidades:**
- Formulário email + senha
- Link "Esqueceu senha?"
- Design centralizado

### **Tela 4 - Cadastro ONG**

**Funcionalidades:**
- Formulário completo (nome, email, telefone, endereço, senha)
- Validação visual dos campos
