# 🐾 PetEncontra – Plataforma de Adoção de Pets

## 📌 Descrição do Projeto
O PetEncontra é uma plataforma web desenvolvida para conectar pessoas interessadas em adotar animais com organizações de resgate e abrigos. O sistema visa facilitar o processo de adoção, aumentar a visibilidade de pets disponíveis e reduzir o abandono de animais.

## 🎯 Problema
Em Fortaleza, muitos animais abandonados não encontram lares devido à falta de visibilidade e organização dos processos de adoção.

## 💡 Objetivo
Criar uma plataforma digital que conecte adotantes e ONGs, facilitando o processo de adoção de forma simples, rápida e segura.

## 🌍 ODS 11 – Cidades Sustentáveis
O projeto contribui para cidades mais sustentáveis ao promover o cuidado animal e reduzir impactos sociais do abandono.

## 🧩 Escopo
- Cadastro de usuários
- Cadastro de pets
- Busca de pets para adoção
- Solicitação de adoção
- Gerenciamento por ONGs

## 🏗️ Arquitetura
Arquitetura baseada em:
- Frontend Web
- Backend API REST
- Banco de Dados Relacional

## 🛠️ Tecnologias
- Frontend: React
- Backend: Node.js
- Banco de dados: PostgreSQL
