# 📋 Requisitos do Sistema

## 👤 Perfis de Usuário
- Visitante
- Usuário cadastrado
- ONG/Administrador

## ✅ Requisitos Funcionais

- RF01: Usuário pode se cadastrar
- RF02: Usuário pode fazer login
- RF03: ONG pode cadastrar pets
- RF04: Usuário pode visualizar pets
- RF05: Usuário pode solicitar adoção
- RF06: ONG pode aprovar/rejeitar solicitações

## ⚙️ Requisitos Não Funcionais

- RNF01: Sistema deve ser responsivo
- RNF02: Tempo de resposta inferior a 2 segundos
- RNF03: Sistema deve garantir segurança dos dados
- RNF04: Disponibilidade de 99%

## 📜 Regras de Negócio

- Apenas ONGs podem cadastrar pets
- Usuários devem estar logados para solicitar adoção
- Cada pet pode ter apenas uma adoção ativa

## 🧠 Histórias de Usuário

- Como usuário, quero buscar pets para adoção
- Como ONG, quero cadastrar pets
- Como usuário, quero solicitar adoção
