# 🏗️ Arquitetura do Sistema

## 📌 Tipo de Arquitetura
Arquitetura cliente-servidor com API REST

## 🧩 Componentes

- Frontend Web (React)
- Backend (Node.js)
- Banco de Dados (PostgreSQL)

## 🔄 Comunicação
- HTTP/HTTPS
- JSON

## 🧱 Padrões utilizados
- MVC
- REST

## 📊 Descrição
O frontend consome APIs do backend, que processa regras de negócio e acessa o banco de dados.

## ⚙️ Justificativa
Essa arquitetura permite escalabilidade, manutenção e separação de responsabilidades.
