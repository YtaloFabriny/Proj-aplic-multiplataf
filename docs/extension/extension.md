## 📌 Problemas Identificados
- Grande quantidade de animais sem adoção  
- Falta de organização digital  
- Dificuldade na gestão de dados  
- Baixa visibilidade para adoção  

## 💡 Solução Proposta
O sistema PetEncontra foi projetado para:

- Centralizar informações de pets  
- Facilitar o processo de adoção  
- Melhorar a comunicação entre adotantes e organizações  

## ✅ Validação
A solução proposta atende às necessidades observadas em abrigos reais, contribuindo para melhor gestão e aumento das chances de adoção.
